# 커피 메뉴 이미지

첨부하신 이미지에서 각 커피 메뉴를 개별 이미지로 저장해주세요.

## 필요한 이미지 파일

1. **espresso.jpg** - 에스프레소 (작은 회색 컵, 진한 크레마)
2. **americano.jpg** - 아메리카노 (투명 유리잔, 증기)
3. **latte.jpg** - 라떼 (베이지색 머그, 하트 라떼 아트)
4. **cappuccino.jpg** - 카푸치노 (회색 컵과 접시, 코코아 파우더)
5. **flat-white.jpg** - 플랫 화이트 (회색 컵과 접시, 단일 하트 아트)
6. **cold-brew.jpg** - 콜드 브루 (투명 유리잔, 얼음)

## 이미지 저장 방법

1. 첨부하신 이미지를 다운로드
2. 이미지 편집 프로그램으로 각 커피를 개별 이미지로 잘라내기
3. 위의 파일명으로 `ui/public/images/` 폴더에 저장

또는 하나의 이미지 파일을 `coffee-menu-grid.jpg`로 저장하고, CSS로 각 영역을 잘라서 사용할 수도 있습니다.
